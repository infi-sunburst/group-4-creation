#### 激光切割注意事项
1. 在使用激光切割机之前要仔细阅读注意事项，避免带来不必要的损伤.
2. 免被激光烧伤——放置材料并调整好位置，盖上机盖，最后打开激光.
3. 避免吸入有害气体——保持室内通风.                        
4. 防火安全——激光切割机工作时不要离开.
5. 激光切割机电流不要过大.
